# Módulo 07: Extensibilidade e Multitenancy

![Infográfico sobre Extensabilidade em CAP Java](./Infograficos/07.02.01_Extensibility_Toolkit.png)

Este módulo aborda tópicos avançados essenciais para a criação de aplicações SaaS (Software as a Service) e para a construção de soluções modulares e reutilizáveis. Aprenderemos como o CAP é projetado para suportar múltiplos clientes (tenants) de forma segura e isolada, e como podemos estender a funcionalidade padrão através de plugins.

## Tópicos Abordados

Explore os conceitos de arquiteturas avançadas através dos seguintes subdiretórios:

- **[01-arquitetura-multitenant-saas](./01-arquitetura-multitenant-saas/README.md):** Domine a arquitetura multitenant do CAP. Entenda como o framework automatiza o provisionamento de tenants, o isolamento de dados e o gerenciamento do ciclo de vida de subscrições de clientes.

- **[02-plugins-e-modularidade](./02-plugins-e-modularidade/README.md):** Descubra como criar extensões reutilizáveis (plugins) que podem ser facilmente adicionadas a outras aplicações CAP. Abordaremos como empacotar modelos CDS e lógica de negócios (Event Handlers) em artefatos Maven independentes.
