# Módulo 07 (Parte 1): Arquitetura Multitenant (SaaS)

![Infográfico sobre Multitenancy em CAP Java](../Infograficos/07.01.01_Dominando_Multitenancy_CAP_Java.png)

Este módulo foca em um dos cenários mais importantes para aplicações empresariais na nuvem: a multitenancy, ou a capacidade de uma única aplicação servir múltiplos clientes (tenants) de forma segura e isolada.
