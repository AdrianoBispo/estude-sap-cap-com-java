# Módulo 01 (Parte 2): Linguagens Core - CDL, CSN e CXN

![Infográfico sobre o ecossistema CDS](../Infograficos/01.02.01_SAP_CDS_Projeto_E_Realidade.png)

Este módulo aprofunda o conhecimento nas linguagens que formam o coração do SAP Cloud Application Programming Model (CAP). Entenderemos como o Core Data Services (CDS) se materializa através de diferentes representações, cada uma com um propósito específico no ciclo de vida do desenvolvimento.
