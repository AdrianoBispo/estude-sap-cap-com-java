# Módulo 01 (Parte 3): Compilador CDS e a API de Reflexão

![Infográfico sobre a API de Reflexão (Modelo Vivo)](../Infograficos/01.03.02_Modelo_Vivo_Reflection_API.png)

Este módulo explora dois componentes cruciais do ecossistema CAP: o compilador CDS, que transforma seus modelos em artefatos consumíveis, e a API de Reflexão, que permite a interação dinâmica com esses modelos em tempo de execução.
