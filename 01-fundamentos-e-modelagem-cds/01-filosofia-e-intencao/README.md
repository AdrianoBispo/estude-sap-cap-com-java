# Módulo 01 (Parte 1): Filosofia e Intenção

![Infográfico sobre Modelagem com Intenção](../Infograficos/01.01.02_Modelagem_com_Intencao.png)

Este módulo foca nos conceitos fundamentais que guiam o desenvolvimento de aplicações com o SAP Cloud Application Programming Model (CAP). Exploramos a "filosofia" por trás do CAP, a importância da modelagem orientada pela intenção e a estrutura do Core Data Services (CDS).
