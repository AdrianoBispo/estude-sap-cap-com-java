# Módulo 01 (Parte 4): Modelagem Performática e Modular

![Infográfico sobre Modelagem para Performance](../Infograficos/01.04.01_Performance_Modeling_for_Modern_CAP_Development.png)

Este módulo foca em técnicas avançadas de modelagem em CDS, abordando como criar modelos que não são apenas semanticamente ricos, mas também performáticos e modulares. A performance começa no design do modelo.
