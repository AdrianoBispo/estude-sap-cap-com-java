# Módulo 04 (Parte 2): Gestão de Rascunhos (Drafts)

![Infográfico sobre a maestria em Drafts com Fiori e Java](../Infograficos/04.02.01_Fiori_Drafts_Java_Mastery.png)

O suporte a rascunhos (drafts) é um requisito comum em aplicações empresariais, permitindo que os usuários salvem seu trabalho em andamento sem validá-lo ou ativá-lo imediatamente. O CAP oferece suporte nativo e transparente para este fluxo, tanto no backend quanto no frontend.
