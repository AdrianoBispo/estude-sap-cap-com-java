# Módulo 04 (Parte 1): Fiori Elements e Anotações de UI

![Infográfico sobre Fiori Elements e Metadados](../Infograficos/04.01.02_Fiori_Elements_Metadados.png)

Este módulo demonstra um dos recursos mais poderosos do CAP: a capacidade de gerar UIs ricas e padronizadas com SAP Fiori Elements a partir de metadados. As anotações `@UI` adicionadas diretamente ao modelo CDS atuam como "dicas" para o framework de UI, definindo a aparência e o comportamento da aplicação.
