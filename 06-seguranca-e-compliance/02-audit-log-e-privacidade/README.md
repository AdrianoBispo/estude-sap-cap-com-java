# Módulo 06 (Parte 2): Audit Log e Privacidade de Dados

![Infográfico sobre Maestria em Audit Logging](../Infograficos/06.02.01_Audit_Logging_Maestria.png)

Além de controlar o acesso, é crucial registrar quem fez o quê e garantir a conformidade com as regulações de privacidade de dados. Este módulo explora como o CAP auxilia nessas tarefas críticas.
