# Módulo 05 (Parte 1): Consumo de Serviços Remotos

![Infográfico sobre o Consumo de Serviços Remotos](../Infograficos/05.01.01_Consumo_Servicos_Remotos.png)

Este módulo foca na integração síncrona com outros sistemas. O CAP simplifica enormemente o consumo de serviços OData externos (como APIs do S/4HANA ou de terceiros) ao tratá-los como se fossem parte do seu próprio modelo de dados.
