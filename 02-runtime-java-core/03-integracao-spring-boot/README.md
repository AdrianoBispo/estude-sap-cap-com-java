# Módulo 02 (Parte 3): Integração com Spring Boot

![Infográfico sobre a integração CAP Java e Spring Boot](../Infograficos/02.03.01_CAP_Java_Meets_Spring_Boot.png)

A maior força do CAP Java é sua integração nativa com o Spring Boot. Este módulo explora como o CAP aproveita o ecossistema Spring para autoconfiguração, injeção de dependências e muito mais, permitindo que os desenvolvedores se concentrem na lógica de negócios.
