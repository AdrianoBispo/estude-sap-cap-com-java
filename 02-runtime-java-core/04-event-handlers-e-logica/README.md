# Módulo 02 (Parte 4): Event Handlers e Lógica de Negócio

![Infográfico sobre o Runtime Java](../Infograficos/02.04.01_Dominando_Runtime_Java.png)

Este é o módulo onde a lógica de negócio da sua aplicação ganha vida. O CAP Java utiliza um modelo de programação reativo e baseado em eventos, onde o seu código é implementado em `Event Handlers` que respondem a diferentes fases do processamento de uma requisição.
