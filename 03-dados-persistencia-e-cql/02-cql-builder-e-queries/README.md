# Módulo 03 (Parte 2): CQL Builder e Queries

![Infográfico sobre a evolução do SQL para o CQL](../Infograficos/03.02.01_CQL_Evolucao_SQL.png)

Este módulo é um mergulho profundo na **CDS Query Language (CQL)**, a forma idiomática e segura de consultar dados no CAP. Em vez de concatenar strings de SQL, você usará uma API fluente e tipada para construir instruções de `SELECT`, `INSERT`, `UPDATE` e `DELETE`, garantindo que suas queries sejam robustas e portáveis entre diferentes bancos de dados.
