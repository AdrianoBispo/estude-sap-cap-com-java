# Módulo 03: Dados, Persistência e CQL

![Infográfico sobre a Arquitetura de Persistência](./Infograficos/03.01.01_Arquitetura_Persistencia.png)

Este módulo foca em como o CAP Java interage com a camada de persistência. Abordaremos a arquitetura de dados, a manipulação de entidades e a construção de consultas poderosas e seguras com a CDS Query Language (CQL).

## Tópicos Abordados

Explore os componentes de persistência e dados através dos seguintes subdiretórios:

- **[01-arquitetura&manipulacao-dados](./01-arquitetura&manipulacao-dados/README.md):** Entenda a arquitetura de persistência do CAP e como manipular dados utilizando as interfaces de acesso (`CdsData`) para maior produtividade e segurança de tipos.

- **[02-cql-builder-e-queries](./02-cql-builder-e-queries/README.md):** Domine a CDS Query Language (CQL), a API fluente para construir consultas de forma declarativa. Aprenda a usar o `CQL Builder` para criar queries complexas e a executá-las com o `PersistenceService`.
